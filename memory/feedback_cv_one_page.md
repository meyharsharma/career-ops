CV must ALWAYS fit on 1 page but fill 80-85% of the page. No exceptions.

**Why:** User explicitly requested this. 2-page CVs are not acceptable. Too much whitespace is also not acceptable.

**How to apply:**
- ALL TEXT BLACK ONLY. No colored text anywhere. Use font size and bold for emphasis only.
- Summary: 2-3 lines max
- Skills: plain inline text with line breaks per category, no fancy tag grid
- Projects: inline title + description, no separate tech stack lines unless space allows. Include all 4 projects.
- Name: centered, contact details centered below the name
- Font sizes: body 11px, bullets 10px, skills 10.5px — sized to fill 90-95% of page
- Section title: black, bold, uppercase, thin black underline (not colored)
- Company names: black bold (not colored)
- Keep exactly 1 page — if it spills, trim bullet descriptions first, then remove project tech lines
